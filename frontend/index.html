note片思いフォローチェッカー

フォローバックされていない人・フォロー返し忘れている人を見つけます。

おすすめ

拡張機能でコピー

note.comにログインしたタブでボタンを押すだけでCookieをコピーできます。

拡張機能をダウンロード（ZIP）
  1. ZIPを展開する
  2. Chromeのアドレスバーに貼り付けて開く
  3. デベロッパーモードをONにする
  4. 「パッケージ化されていない拡張機能を読み込む」から展開したフォルダを選ぶ
手動

開発者ツールからコピー

  1. note.comにログインした状態で開発者ツールを開く（Mac: Cmd+Opt+I）
  2. Networkタブを開いてnote.comのページをリロードする
  3. 一覧のどれかのリクエストをクリックする
  4. Request Headers の cookie: の値を丸ごとコピーする

注意: このCookieはnote.comアカウントの操作権限そのものです。信頼できない相手には絶対に渡さないでください。 このアプリでは入力欄に保存せず、フォロー状態の確認とフォロー操作の通信にその場で使うだけで、ディスクや外部には送信・保存しません。

メモ: アカウントチェックにはCookieが必要です。ログイン中の本人としてフォロー済みかを確認します。 一度に多くのフォロー/フォロー解除を行うと、note.com側のレート制限（429エラー）にかかることがあります。

⚠️ フォロー/フォロワーが多いアカウントのため、note.com側の上限により一部しか確認できていない可能性があります。
